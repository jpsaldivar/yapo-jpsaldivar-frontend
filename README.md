# Web app

## Instalación 

```bash
$ yarn install
```
or 

```bash
$ npm install
```

## Running the app

With Yarn 
```bash
# development
$ yarn start

# or with npm
$ npm run start
```

## Acceder a la app

Ir a [http://localhost:3006](http://localhost:3006)
